Diaclopedia

Diablo II: Resurrected Item Encyclopedia

About

Diaclopedia is an item encyclopedia app for Diablo II: Resurrected. Browse and search detailed stats for weapons, armor, unique items, set items, runewords, and runes — all in one place.

Features

  • Search items by name or attributes
  • Browse unique items, set items, and runewords
  • Runeword combination lookup
  • Attribute-based item search
  • Available on Android and iOS
